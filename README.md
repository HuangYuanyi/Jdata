# 基本解决方案
本题要预测两个部分，s1为购买的用户，s2为用户购买的时间
# 解题思路
这个题目首先要预测的是购买的用户，只有s1比较高的情况下，s2自然而然的比较高。因此我们觉得这题可以从s1入手，先预测出购买的用户，再来预测s2用户购买的时间。
